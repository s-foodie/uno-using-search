# uno-using-search
UNO using Minimax and Expectimax
This project implements a simplified UNO game with 3 players using adversial search algorithms

Players:
Player 1 - Minimax (defensive)
Player 2 - Expectimax (offensive)
Player 3 - Manual or AI simulation

Features:
Minimax Search of depth 3
Expectimax Search of depth 3
Skip card logic
Decision making based on evaluation functions
